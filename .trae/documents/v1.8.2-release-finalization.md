## Proposed Changes

### 1. 修复 ErrorBoundary tsc 错误 — `src/components/ErrorBoundary.tsx`

在 class body 中添加 `declare props: Props;`（放在 `state` 声明之前）：

```tsx
export class ErrorBoundary extends Component<Props, State> {
  declare props: Props;  // 解决 useDefineForClassFields: false 下 this.props 不识别
  state: State = { hasError: false, error: null };
  // ... 其余不变
}
```

**Why**: `declare` 关键字告诉 TypeScript 该属性存在（继承自基类）但不发射运行时代码。这是 `useDefineForClassFields: false` 配置下 React class 组件的标准修复方式，不影响运行时行为。
