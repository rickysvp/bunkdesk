Guest 接口扩展 + 新增枚举 + 向后兼容

Guest 接口变更

// 新增字段（Optional，向后兼容）
firstName?: string; // 拆分后的名
lastName?: string; // 拆分后的姓
phone?: string; // 已有
email?: string; // 已有
idType?: IdDocumentType; // 新增枚举
arrivalTime?: ArrivalSlot; // 新增枚举
referral?: string; // 推荐来源 / 自由文本
// 新枚举：证件类型
type IdDocumentType = "passport" | "idCard" | "driverLicense";
// 新枚举：抵店时段
type ArrivalSlot = "morning" | "afternoon" | "evening" | "late";
// 保留 name 字段（向后兼容）
name: string; // 全名，新数据 first+last 拼接，旧数据保留

优点

  • 所有新字段 optional，旧 localStorage 数据无需迁移
  • firstName/lastName 不覆盖 name，向后兼容 iCal / 现有 seed
  • 枚举类型利于 i18n 显示和筛选

权衡

  • name 与 first/last 会有冗余（以 first/last 为准，name 自动拼接）
  • 旧数据 name 展示不变，新数据优先展示 first + last
