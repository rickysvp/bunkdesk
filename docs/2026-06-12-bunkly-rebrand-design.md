# Bunkly 品牌重塑 + 轻量级体验降维 设计文档

## 竞争策略

BunkFlow 做重（AI+OTA+自动化），Bunkly 做轻（零门槛、即开即用、极简操作）。

核心叙事：
- BunkFlow = 未来的 AI 管家（期货，Q2-Q3 2026）
- Bunkly = 今天的运营利器（现在就能用，3 秒上手）

## 变更范围

### 1. 品牌更名 HostelOps → Bunkly
- 所有代码中的 "HostelOps" 文本替换为 "Bunkly"
- Logo "HO" → "Bk"
- 页面标题、i18n key、注释中的品牌名
- index.html title 更新

### 2. 落地页重写
对标 BunkFlow 的痛点叙事但主打轻量：

**Hero 区域：**
- 标题："Check in a guest. In one tap." / "一键入住，一步到位"
- 副标题：强调轻量 vs BunkFlow 的复杂
- Badge："Free forever · No setup" / "永久免费 · 零配置"

**痛点区域（对标 BunkFlow 的 "This is what's killing your hostel business"）：**
- "Your PMS needs a PMS" — 现有系统太复杂
- 数据化：8 screens vs 1 tap, 30 min setup vs 3 sec

**Before/After 对比：**
- Before: 打开 PMS → 找房间 → 填表单 → 分配 → 确认（8 步）
- After: 看到空床 → 点一下 → 完成（1 步）

**功能展示（6 卡片，保持现有但调整文案强调轻量）：**
1. Visual Bed Board → "See it. Tap it. Done."
2. Fast Check-In → "60 seconds, not 3 minutes"
3. Shift Handover → "Auto-logged. Zero effort."
4. Calendar & Reservations → "iCal import. One click."
5. Task Board → "Assign. Track. Done."
6. Activity Management → "Organize. Sign up. Go."

**竞品对比表：**
| | BunkFlow | Bunkly |
|---|---|---|
| Setup | 30 minutes | 3 seconds |
| Learning curve | Days | Zero |
| Price | ₹1,499/mo | Free |
| Status | Waitlist (2026) | Try now |
| AI required | Yes | No |
| Bed-level inventory | Yes | Yes |

**CTA：** "Try Bunkly Now — Free" / "立即体验 Bunkly — 免费"
